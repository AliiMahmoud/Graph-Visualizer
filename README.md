## Graph Visualizer

is a Java Program to apply some algorithms on a visualized graph
  - Simple Graph
    - Directed
    - Undirected
  - Multigraph
    - Directed
    - Undirected

## Installation

 - Minimum Java SE Runtime Environment : 5.0
